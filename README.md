# 1. Install Oh My Zsh
```bash
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
```

# 2. Clone this repo
```bash
cd ~
git clone https://github.com/bmichalkiewicz/dotfiles
cd ~/dotfiles
```

# 3. Run setup scripts
```bash
./install.sh
devbox shell
./sync.sh
```
